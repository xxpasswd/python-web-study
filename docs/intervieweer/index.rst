从请求到响应
++++++++++++++

1. 浏览器解析url，取出host等其他信息
2. 先从本地host查询ip地址，没有从dns服务器上查询
3. 浏览器封装http请求信息，发送至目标服务器
4. 应用服务器接受到请求，将请求分发给web服务器
5. web服务器解析请求，将请求路由到视图函数
6. 视图函数进行处理，返回相应
7. web服务器封装响应，返回给应用服务器
8. 应用服务器将请求返回给浏览器
9. 浏览器进行渲染输出
